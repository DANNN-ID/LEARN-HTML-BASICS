Learning List (Bullets, Numbering, Definition, & Details)

1. List Bullets in HTML

Dalam HTML terdapat beberapa macam bullets yang bisa digunakan, yaitu dengan memasukkan tipe bullets ke dalam nilai atribut type pada tag ul. Beberapa tipe bullets yang bisa digunakan adalah disc, square, dan circle. Untuk contoh penggunaannya bisa kalian lihat di bawah ini.

Contoh List Bullet dengan Tipe Disc :

  • List 1
  • List 2
  • List 3

Contoh List Bullet dengan Tipe Square :

  • List 1
  • List 2
  • List 3

Contoh List Bullet dengan Tipe Circle :

  • List 1
  • List 2
  • List 3

2. List Numbering in HTML

Dalam HTML terdapat beberapa macam numbering yang bisa digunakan, yaitu dengan memasukkan tipe numbering ke dalam nilai atribut type pada tag ol. Beberapa tipe numbering yang bisa digunakan adalah 1, A, a, I (romawi besar), dan i (romawi kecil). Untuk contoh penggunaannya bisa kalian lihat di bawah ini.

Contoh List Numbering dengan Tipe 1 :

  1. List 1
  2. List 2
  3. List 3

Contoh List Numbering dengan Tipe A :

  1. List 1
  2. List 2
  3. List 3

Contoh List Numbering dengan Tipe a :

  1. List 1
  2. List 2
  3. List 3

Contoh List Numbering dengan Tipe I (Romawi Besar) :

  1. List 1
  2. List 2
  3. List 3

Contoh List Numbering dengan Tipe i (Romawi Kecil) :

  1. List 1
  2. List 2
  3. List 3

4. Details List in HTML

Detail list biasa digunakan sebagai pemberi definisi pada list itu sendiri. Mungkin tujuannya hampir sama dengan penggunaan definition list, yaitu memberikan definisi pada list tertentu. Namun penggunaan detail list ini lebih unik daripada definition list karena kamu harus meng-click list itu sendiri untuk menampilkan definisinya.
